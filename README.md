# live-code-catcher

> 운빨존많겜 유튜브 라이브 시청자 참여(시참) 초대 코드 자동 입력 프로그램

유튜브 라이브 방송 화면에 송출되는 **초대 코드를 자동으로 인식**하여 운빨존많겜에 입력해주는 자동화 도구입니다.  
더 이상 손으로 숫자를 보고 타이핑할 필요 없이 **F9 한 번**으로 끝납니다.

---

## 주요 기능

- 화면에 표시된 초대 코드 박스를 이미지 인식으로 자동 탐색
- AI OCR(ddddocr)로 게임 폰트 숫자 인식
- 입력창 자동 클릭 → 붙여넣기 → Enter → 참가하기 버튼 클릭까지 완전 자동화
- 창 크기가 변해도 멀티스케일 매칭으로 대응

---

## 설치

```bash
pip install -r requirements.txt
```

### requirements
- `ddddocr` - 게임 UI 스타일 숫자 OCR
- `opencv-python` - 이미지 템플릿 매칭
- `pyautogui` - 마우스/키보드 자동화
- `pyperclip` - 클립보드 제어
- `pynput` - 마우스 이벤트
- `keyboard` - 단축키 처리
- `Pillow` - 화면 캡처 및 이미지 처리

## 사용 방법

```bash
python auto_number.py
```

운빨존많겜을 실행하고 유튜브 라이브 화면에 초대 코드가 표시되면 **F9** 를 누르세요.

### 단축키

| 키 | 기능 |
|----|------|
| `F2` | 초대 코드 인식 테스트 (인식된 숫자를 클립보드에 저장) |
| `F3` | 입력창 탐색 테스트 |
| `F4` | 참가하기 버튼 탐색 테스트 |
| `F9` | **전체 자동 실행** (인식 → 입력 → 참가) |
| `ESC` | 종료 |

---

## 처음 사용 시 테스트 순서

1. 초대 코드가 화면에 뜬 상태에서 `F2` → 콘솔에 숫자 출력되는지 확인
2. 입력창이 열린 상태에서 `F3` → 좌표 출력되는지 확인
3. 참가하기 버튼이 보이는 상태에서 `F4` → 좌표 출력되는지 확인
4. 모두 정상이면 `F9`로 실전 사용

인식이 안 될 경우 `debug/raw.png`에서 실제 캡처된 영역을 확인하세요.
