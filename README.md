# live-code-catcher

> 운빨존많겜 유튜브 라이브 시청자 참여(시참) 초대 코드 자동 입력 프로그램

유튜브 라이브 방송 화면에 송출되는 초대 코드를 자동으로 인식해 운빨존많겜에 입력해주는 자동화 도구입니다.  
**F9 한 번**으로 코드가 뜨는 순간 자동 입력 → 참가까지 완료됩니다.

---

## 설치

```bash
pip install -r requirements.txt
```

---

## 처음 사용 시 설정 (최초 1회)

### 1단계 — 초대 코드 위치 등록 (F7)

1. 유튜브 라이브에서 초대 코드가 화면에 표시된 상태 만들기
2. 마우스를 **초대 코드 숫자 위에 올리고** `F7` 누르기
3. 박스 크기를 자동 감지해 `config.json`에 좌표 저장
4. 창 크기 바꾸면 F7로 재등록 필요

### 2단계 — 인식 테스트 (F2)

초대 코드가 화면에 표시된 상태에서 `F2`를 눌러 숫자가 정상 인식되는지 확인합니다.  
성공하면 인식된 숫자가 클립보드에 자동 복사됩니다.

### 3단계 — 입력창 / 참가 버튼 테스트 (F3, F4)

- `F3` — 게임 코드 입력창이 열린 상태에서 눌러 탐색 확인
- `F4` — 참가하기 버튼이 보이는 상태에서 눌러 탐색 확인

---

## 실전 사용

```bash
python auto_number.py
```

초대 코드가 뜨기 전에 **F9**를 눌러두면 코드가 감지되는 순간 자동으로 입력 → 참가 완료 후 프로그램이 종료됩니다.  
`F9`를 다시 누르면 중지됩니다.

---

## 단축키 정리

| 키 | 기능 |
|----|------|
| `F7` | 초대 코드 위치 등록 (숫자 위에 마우스 올리고 누르기) |
| `F2` | 숫자 인식 테스트 + 클립보드 저장 |
| `F3` | 입력창 탐색 테스트 |
| `F4` | 참가하기 버튼 탐색 테스트 |
| `F9` | 자동 대기 시작 / 중지 |
| `ESC` | 프로그램 종료 |

## 파일 구성

```
live-code-catcher/
├── auto_number.py        # 메인 프로그램
├── requirements.txt      # 패키지 목록
├── config.json           # F7로 저장된 좌표 (자동 생성)
├── images/
│   ├── input.png         # 코드 입력창 템플릿
│   └── join.png          # 참가하기 버튼 템플릿
└── debug/
    └── raw.png           # 인식 테스트 캡처 확인용 (자동 생성)
```

---

## 주의사항

- 유튜브 창 크기를 바꾸면 **F7로 재등록** 필요
- `images/input.png`, `images/join.png`는 본인 게임 UI에 맞게 교체 가능
- 인식이 안 될 경우 `debug/raw.png`에서 캡처 영역 확인
